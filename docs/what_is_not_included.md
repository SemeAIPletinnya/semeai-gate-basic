# What Is Not Included

This basic open-source package intentionally excludes:

- private SemeAI memory workspace;
- Twitter/X archive;
- local Year 1 archive;
- desktop/Tauri UI;
- receipts corpus from local development;
- private Silence-as-Control / SaC runtime dependencies;
- local Ollama integration;
- private product docs and audit PDFs;
- generated build artifacts;
- cloud/SaaS runtime.

The goal is a small, readable release-control contract for LLM answers.
